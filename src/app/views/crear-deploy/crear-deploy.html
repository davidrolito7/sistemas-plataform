Guia Angular

Sigue estos pasos antes de generar el ZIP.

  1. PASO 1 Agrega web.config en src/

    Descarga el archivo y cambia la URL por el nombre de tu proyecto:

    <action type="Rewrite" url="/tu-proyecto/" />
  2. PASO 2 angular.json → assets

    Dentro del array assets de tu build, agrega: 

    {
  "glob": "web.config",
  "input": "src",
  "output": "/"
}
  3. PASO 3 Build y ZIP

    Ejecuta el build:

    ng build

    Comprime los archivos dentro de browser/, no la carpeta en si.
